## 1. 목표

`index.html` 히어로 슬라이드 버튼 클릭 시 `pages/coins.html?q=...`을 경유하던
방식을 `coins/coin-{slug}.html` 직접 이동으로 전환. Google Sheets fetch 대기
동안 SSG 카드 20개가 먼저 노출되는 깜빡임 현상 제거.

  `slug` 필드로 교체. 버튼 href가 `coins/coin-{slug}.html` 직접 링크로 생성됨.

### href 생성 로직 (renderHeroSlides 함수 내)
```js
const href = s.slug
  ? `coins/coin-${s.slug}.html`           // slug 있으면 직접 이동
  : `pages/coins.html?q=${encodeURIComponent(s.keyword || '')}`;  // 폴백
```
