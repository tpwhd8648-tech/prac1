# 뱃지 확인 및 모바일 수정 인수인계 문서

작성일: 2026-06-22

## 1. 목표

이전 세션에서 구현한 당일수령 뱃지를 실제 사이트에서 검증하고,
모바일에서 뱃지가 카드 텍스트와 겹치는 버그를 수정했다.

## 2. 현재 상태

**완료 (push 후 원격 반영 확인)**

- 당일수령 뱃지 브라우저 확인 (사용자 직접 확인)
  - coins.html, index.html, coin-detail.html 모두 정상 노출 확인
  - 9개 상품(메이플, 브리타니아, 캥거루, 버팔로, 이글, 필하모닉, 크루거, 판다,
    코뿔소)에만 우상단 골드 톤으로 정상 노출 확인 완료

- 모바일 뱃지 겹침 버그 수정 (커밋 `76af496`)
  - 증상: 480px 이하 모바일에서 2열 카드 레이아웃이 좁아지며 뱃지 텍스트가
    카드 우상단 텍스트와 겹침
  - 수정: `style.css` `@media (max-width: 480px)` 블록에 `.badge-sameday` 추가
    - 폰트 `11px → 9px`, 패딩 `3px 9px → 2px 5px`, 위치 `top/right 10px → 6px`,
      자간 제거

**확인 완료 (종결)**

- 말띠(order=14) / 네스호(order=13) 순서 — 의도된 순서임. 변경 불필요.

## 5. 참고 사항

- 뱃지 관련 CSS는 `style.css`의 `.badge-sameday`(기본)와
  `@media (max-width: 480px)` 블록(모바일 축소) 두 곳에 나뉘어 있음.
  추후 뱃지 스타일 수정 시 두 곳 모두 확인할 것.

- 당일수령 뱃지 노출 조건: `available=TRUE` AND `same_day=TRUE`.
  품절 상품(`available=FALSE`)은 SOLD OUT 오버레이가 우선되어 뱃지 숨김.
