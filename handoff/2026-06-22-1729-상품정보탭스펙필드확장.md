# 상품정보 탭 스펙 필드 확장 인수인계 문서

작성일: 2026-06-22

## 1. 목표

코인 상세페이지 [상품정보] 탭의 스펙 테이블을 확장해, 기존 4개 필드(발행처/발행국/순도/중량)에
연도·직경·두께·상태를 추가하고, 불필요한 미확인 안내 문구를 제거한다.

## 2. 현재 상태

**완료 (push 반영 확인)**

- `coin-descriptions.js` — 22개 상품 전체 스펙에 아래 3개 필드 추가:
  - `year`: 전체 2026 (레이디저스티스만 2025)
  - `diameter`: 확인된 11개 수치 입력, 나머지 11개 "확인 중"
    - 확인된 값: 메이플 30mm / 브리타니아·버팔로·퀸즈라이언·레이디저스티스 32.69mm /
      캥거루·아웃백 32.1mm / 아메리칸 이글 32.7mm / 필하모닉 37mm /
      크루거랜드 32.77mm / 판다 32mm
  - `thickness`: 확인된 11개 수치 입력, 나머지 11개 "확인 중"
    - 확인된 값: 메이플·브리타니아·버팔로·아메리칸 이글·퀸즈라이언·레이디저스티스 2.87mm /
      캥거루·아웃백 2.65mm / 필하모닉 2.37mm / 크루거랜드 2.84mm / 판다 2.7mm
  - `condition`: 전체 'Brilliant Uncirculated'
  - 직경·두께 미확인 상품: 코뿔소, 성조지, 라이언이글, 말띠, 네스호, 스완,
    체코라이언, 케이브라이언, 로얄드래곤, 브리티시라이언, 세인트조지드래곤

- `coin-detail.html` — 상품정보 탭 테이블에 연도/직경/두께/상태 행 추가
  - 표시 순서: 연도 → 발행처 → 발행국 → 순도 → 중량 → 직경 → 두께 → 상태
  - `specs.diameter`, `specs.thickness` 없으면 자동으로 "확인 중" 표시

- `coin-detail.html` — 미확인 스펙 안내 문구 및 관련 CSS 제거
  - 제거 항목: `* 위 발행처/순도 정보는 최종 확인 전 추정치입니다...` 문구
  - 제거 항목: `.spec-unverified-note` CSS 규칙

- `docs/PROJECT_INSTRUCTIONS.md` — 할일 확인/인수인계 작성 시 최근 3개 핸드오프
  교차 확인 규칙 추가 (섹션 1·4)

- 최종 커밋: `52b2b20`

## 5. 참고 사항

- **스펙 필드 구조 (현재)**: `year`, `mint`, `country`, `purity`, `weight`, `diameter`, `thickness`, `condition`, `verified`
  - 신규 필드 추가 시 `coin-descriptions.js`만 수정하면 되고, `coin-detail.html`은 `|| '확인 중'` 폴백으로 자동 처리됨

- **신규 상품 추가/삭제 시**: `coin-descriptions.js`의 `COIN_DESCRIPTIONS` 배열에만 항목 추가/삭제.
