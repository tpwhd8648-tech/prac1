# 우선순위 재조정 인수인계 문서

작성일: 2026-06-19

## 1. 목표
사이트 운영 중 남은 작업(기술적 SEO 개선, 카테고리 배너 이미지 교체, 검색 노출 확인, 기타 선택사항)을 정리하고 진행 순서를 확정한다. 실제 작업 착수는 아직 이루어지지 않았다.

## 2. 현재 상태

### 확인 완료
- 4개 작업 항목 모두 미착수 상태임 (코드/레포 변경 없음).
- 최종 확정된 진행 순서는 아래 "진행 단계" 및 To-Do 참고.

### 미완료
- 1순위(기술적 SEO 개선)부터 시작할지 사용자 확인 대기 중.

## 3. 진행 단계

다음 세션에서 이어갈 경우:
1. (필수) 사용자에게 1순위(기술적 SEO 개선) 작업을 바로 시작할지 확인한다.
2. (필수) 1순위 착수 시: 레포 클론 → 각 페이지(html)의 title/description 메타태그 점검 → 상품 이미지 alt 텍스트 누락 여부 grep → 이미지 로딩 속도 관련 항목(용량, lazy-loading 적용 여부 등) 점검 순으로 진행한다.
3. (선택) 1순위 완료 후 2순위(카테고리 배너 이미지 교체)로 이동 시, 사용자에게 v1/v2 시안에서 구체적으로 어떤 점이 아쉬웠는지(색감/모티프/구도 등) 먼저 질문한다. 막연한 피드백만 받고 시안을 또 만들면 같은 결과가 반복될 가능성이 높다는 점을 이전 세션에서 확인함.

## 4. 다음에 진행할 일 (To-Do)

**1순위 — 기술적 SEO 개선**
- [ ] 1-1. 각 페이지 title/description 메타태그에 검색 키워드 보강
- [ ] 1-2. 상품 이미지에 alt 텍스트 추가
- [ ] 1-3. 이미지 로딩 속도 점검 및 최적화

**2순위 — 카테고리 배너 이미지 교체**
- [ ] 2-1. 어떤 점이 왜 아쉬웠는지 구체적으로 합의 (색감/모티프/구도 등)
- [ ] 2-2. 3번째 시안 제작 → 승인 후 style.css/index.html 반영 → push

**3순위 — 수집 현황 확인**
- [ ] 3-1. 네이버 서치어드바이저에서 페이지 수집 현황 확인
- [ ] 3-2. 네이버에서 사이트명/키워드로 직접 검색해서 노출 여부 확인

**4순위 — 선택사항**
- [ ] 4-1. 구글 서치 콘솔 등록
- [ ] 4-2. 콘텐츠 마케팅/백링크 확보
- [ ] 4-3. 네이버 파워링크(유료 광고) 검토

## 5. 참고 사항

- **카테고리 배너 이미지 작업(2순위) 관련 기존 메모**: .cat-silver 클래스인데 실제 표시 문구는 "금 시세"라 클래스명-문구가 불일치하는 상태 (별개 이슈, 미수정). Claude는 포토리얼 이미지 생성 도구가 없으므로 SVG/CSS 그래픽만 가능함.
- **레포 파일 변경 없음**: 이번 세션은 순서 조정 논의만 진행했고 git에 push할 변경사항이 없다. 다음 세션 시작 시 별도 git pull/동기화 불필요.
- **GitHub 토큰 미수신**: 이번 세션에도 토큰을 받지 못해 이 문서를 레포에 직접 push하지 못했다. 토큰을 주시면 handoff/ 폴더에 커밋·push할 수 있다.
