## 1. 목표

coins.html에 "당일수령" 탭 필터를 추가하고,
코인 상세페이지의 "당일수령 가능 금화" 섹션에 "전체 보기 ›" 링크를 연결해
방문자가 IN STOCK 코인 목록 전체로 한 번에 이동할 수 있도록 한다.

## 2. 현재 상태

### 2-1. 구현 완료 (이번 커밋에 포함)

- **coins.html — 당일수령 탭 버튼 추가**
  - 기존 "금화" 탭 옆에 "당일수령" 탭(`data-filter="instock"`) 추가
  - `applyTabFilter('instock')`: `.badge-instock` 요소를 가진 카드만 표시, 나머지 hide
  - `?instock` URL 파라미터 감지 시 자동으로 탭 활성화 + 필터 적용

- **scripts/generate-coin-pages.js — 전체 보기 링크 추가**
  - 당일수령 가능 금화 섹션 헤더를 flex row로 변경 (`related-coins-header`)
  - 제목 우측에 `<a href="coins.html?instock">전체 보기 ›</a>` 추가
  - `.related-coins-header`, `.related-coins-more`, `.related-coins-more:hover` CSS 추가

## 5. 참고 사항

- **필터 방식**: `badge-instock` 클래스 유무로 판단. SSG 빌드 시점 기준이므로
  실시간 재고와 1일 최대 차이 날 수 있음 (Actions 00:00 KST 자동 빌드).
- **?instock 파라미터**: 값 없이 키가 존재하면(`urlParams.has('instock')`) 동작.
  `coins.html?instock` 또는 `coins.html?instock=1` 모두 작동.
- **"전체 보기 ›" 링크 시점**: generate-coin-pages.js 수정이므로
  Actions 빌드 후 coin-*.html에 반영됨. 수동 빌드 필요 시 PAT 제공 후 요청.
