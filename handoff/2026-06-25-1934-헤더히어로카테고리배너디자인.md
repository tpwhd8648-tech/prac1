# 헤더/히어로/카테고리 배너 디자인 방향 인수인계 문서

작성일: 2026-06-25

## 1. 목표

헤더/히어로/카테고리 배너 디자인 방향(A안 화이트·B안 다크·C안 풀블랙) 확정 작업 중,
헤더는 확정·반영되었으나 히어로와 카테고리 배너는 아직 방향이 정해지지 않아
다음 세션에서 이어서 진행한다.

## 2. 현재 상태

- **헤더**: A안(화이트)으로 확정되어 코드에 반영됨.
  `nav.js`에서 흰 배경 + `color:#333` 톤 확인 (이번 세션에서 코드로 직접 확인 완료).
- **카테고리 배너**: 미정. 코드(`index.html`의 `.category-banners`, `style.css`의
  `.cat-gold`/`.cat-silver`/`.cat-collectible`)에는 아직 예전 placeholder
  (Unsplash 이미지 + 금색/은색/녹색 그라데이션, 금화·금시세·문의 3종 배너)가
  그대로 남아있음 (확인 완료). A/B/C 시안이 적용된 흔적 없음.
- **히어로**: 미정. `index.html`의 `.hero-slider` 슬라이드 5개(성조지·아메리칸이글·
  버팔로·캥거루·필하모닉)는 기존 구조 그대로이며, A/B/C 시안 적용 흔적 없음.
- 사용자가 "카테고리랑 히어로는 해야해"라고 명시 → 이 두 영역만 다음 세션 작업 대상.
- 이번 세션은 코드 변경 없음. 레포 코드는 커밋 `2ae99f6` 상태 그대로.

## 3. 진행 단계

1. (필수) 카테고리 배너·히어로 디자인 방향(A/B/C 중 또는 별도) 확정 질문 — 헤더와
   통일할지, 다른 톤으로 갈지부터 질문.
2. (필수) 방향 확정 후 `index.html`의 `.category-banners`, `.hero-slider` 및
   `style.css`의 관련 클래스(`.cat-gold` 등) 수정.
3. (필수) 카테고리 배너 placeholder Unsplash 이미지 교체 — 디자인 방향 확정과 함께 처리.
