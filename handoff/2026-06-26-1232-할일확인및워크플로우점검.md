## 5. 참고 사항

- **캐러셀 터치 로직 핵심**: `touch-action`은 CSS 비상속 속성. 터치가 실제로 닿는 자식 요소(`.product-card`)에 직접 `pan-y` 적용 필요. 또한 `passive:false` touchmove 리스너와 `touch-action:pan-y`가 같은 요소에 공존하면 브라우저가 네이티브 스크롤 최적화를 비활성화함 → vertical 확정 즉시 `touchAction = 'auto'`로 전환하는 방식으로 해결.
- **솔드아웃 관련 코드 분산 위치** (수정 시 4곳 항상 함께 점검):
  1. `scripts/generate-coin-pages.js` (SSG 템플릿)
  2. `script.js` (index.html 런타임 가격 갱신)
  3. `coins.html` 인라인 스크립트 (런타임 가격 갱신 + 검색 재렌더링)
  4. `products.js` (coins.html 검색 시 카드 렌더 함수)
