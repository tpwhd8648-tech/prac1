## 2. 현재 상태

### 완료된 작업 (이번 세션, 커밋 순)

**(1) 모바일 햄버거 버튼 오른쪽 정렬 (커밋 `ebc9253`)**
- 원인: 모바일에서 `.header-search`가 `display:none`이 되면서 로고 + 햄버거만 남는데,
  `gap:20px` flex 배치라 햄버거가 중앙으로 쏠렸음
- 수정: `style.css` 768px 미디어쿼리에 `.header-icons { margin-left: auto; }` 한 줄 추가

**(2) 솔드아웃 상품 가격 표시 '품절' → '-' 변경 (커밋 `d508d96`)**
- `scripts/generate-coin-pages.js` 템플릿 3곳 수정
- 코인 페이지 20개 인라인 스크립트 일괄 sed 교체
- 상세 페이지 "현재 품절된 상품입니다" 배너는 사용자 요청으로 **그대로 유지**

**(3) 모바일 캐러셀/브랜드그리드 수직 스크롤 방해 수정 (커밋 `a116259`)**
- 원인: `overflow-x: auto` 컨테이너에서 손가락이 약간 비스듬히 움직이면
  캐러셀이 스크롤 이벤트를 가져가 페이지 스크롤이 안 되는 현상
- 수정: `index.html`의 `#products-scroll-wrap`과 `.brand-grid` 둘 다
  `touch-action: pan-y` 추가

**(4) 솔드아웃 버튼 스타일 일반 버튼과 동일하게 변경 (커밋 `9506277`)**
- `style.css`의 `.btn-soldout`을 `.btn-buy`와 동일한 금색 스타일로 교체
- 기존: 어두운 배경(`#2a2a2a`) + 회색 텍스트(`#555`) → 변경: 금색 배경(`#C8A84B`) + 흰 텍스트

## 5. 참고 사항
