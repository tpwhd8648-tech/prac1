## 5. 참고 사항

- **JS/CSS 수정 시 경로 주의**: 이제 모든 JS/CSS는 `assets/js/`, `assets/css/` 하위에 있음.
  루트에서 직접 편집하던 습관이 있다면 경로 변경 인지 필요.
- **SSG(generate-coin-pages.js)의 이중 경로**: HTML 템플릿에 삽입하는 `<script src>` 경로와,
  Node.js가 파일을 직접 `require`하는 경로가 별개로 존재함. 향후 JS 파일 경로 변경 시
  두 곳 모두 수정 필요 (46번 줄, 50번 줄).
- **workflow trigger paths**: `assets/js/coin-descriptions.js`, `assets/js/coin-data.js`,
  `assets/js/products.js` 중 하나를 수정하면 SSG 자동 재생성 트리거됨.
- **솔드아웃 관련 코드 분산 위치** (수정 시 4곳 항상 함께 점검):
