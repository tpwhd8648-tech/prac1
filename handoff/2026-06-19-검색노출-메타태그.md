## 5. 참고 사항

- 상품 이미지 alt 텍스트는 이미 코드 레벨에서 자동 처리되고 있으므로, 향후 "alt 텍스트 추가" 작업을 다시 검토할 필요 없음 (단, 새로운 이미지 타입을 추가하는 코드를 짤 때는 `alt` 속성을 빠뜨리지 않도록 주의).
- description 메타태그는 화면에 보이지 않고 검색결과 노출에만 영향을 주는 작업이라, 적용 후에도 사이트 디자인/동작에는 변화가 없음 — 정상.
- 이번 작업은 사용자가 토큰을 제공하고 명시적으로 "푸쉬해줘"라고 확인한 후에만 push 진행함 (push 전 변경사항 요약 보고 절차 준수).
- 토큰은 세션 내에서만 사용했고 별도 저장하지 않음.
