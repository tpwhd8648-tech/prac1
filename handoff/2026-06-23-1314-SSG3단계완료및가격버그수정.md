SSG 3단계(coins.html 링크 교체)를 완료하고, 자동 생성된 코인 정적 페이지에서

## 2. 현재 상태

**완료 (push 반영 확인)**

- **GitHub Actions 워크플로우 실제 실행 확인** (커밋 `0aa6b4f` — github-actions[bot]):
  20개 코인 정적 페이지 전부 생성됨. skip된 코인 없음. IMAGE_MAP 20개 전부
  매칭 성공.

- **coins.html 카드 링크 교체** (커밋 `e64fb11`):
  - `getSlugForProduct()` 함수 추가 — IMAGE_MAP 참조해 상품명 → slug 변환
  - `linkTemplate`을 `coin-{slug}.html`로 변경
  - slug 매칭 실패 시 `coin-detail.html?name=...` fallback 유지

- **코인 정적 페이지 가격 미표시 버그 수정** (커밋 `2d7c285`):
  - **원인**: `nav.js`는 시세 fetch 완료 후 `updateCardPricesFromSheet(krwPerOz)`를
    호출해 가격을 갱신하는데, 자동 생성 페이지에 해당 함수가 없어
    nav.js와 연동이 끊겨 있었음. `loadPriceAndStock()` 단독 실행 시점엔
    `tb-gold`/`tb-rate`가 아직 비어있어 가격 계산 불가.
  - **수정**: `_coinPremium` 변수 + `updateCardPricesFromSheet()` 함수 추가.
    nav.js 콜백 연동 및 `loadPriceAndStock()` 내 즉시 계산 모두 이 함수로
    통일. 코인 페이지 20개 전체 + `generate-coin-pages.js` 스크립트 반영.
  - **검증**: 브라우저에서 `coin-kangaroo-2026.html` 가격 표시 확인 완료.

- **mypage.html 메타태그** — 이미 완료 상태였음 (`8e800c0` 커밋에서 반영).
  canonical/og/favicon 전부 정상 확인.

- **coin-detail.html 처리 방향 결정** — **fallback으로 유지**로 확정.
  - 이유: 새 코인 추가 후 Actions 실행 전 공백기 안전망, 외부 링크/북마크 404
    방지, 유지 비용 거의 없음.
  - coins.html의 slug 매칭 실패 시 자동으로 coin-detail.html?name=...으로
    fallback되는 코드는 이미 적용돼 있어 추가 작업 없음.

## 5. 참고 사항

- **SSG 전체 구조 완성**: 코인 정적 페이지 생성(Actions) + coins.html 링크
  연결 + 가격 연동까지 모두 동작 확인. 앞으로 신규 코인 추가 시:
  `coin-descriptions.js` + `products.js`의 IMAGE_MAP + 시트 행 3곳 추가하면
  다음 Actions 실행 시 자동으로 `coin-{slug}.html`까지 생성됨.
- **가격 연동 원리**: 코인 정적 페이지의 가격은 `nav.js`의
  `updateNavPrices()` → `updateCardPricesFromSheet(krwPerOz)` 콜백으로
  갱신됨. `_coinPremium`이 null이면 (시트 로딩 전) 함수가 아무것도 안 하므로
  순서 무관하게 안전하게 동작.
- **coin-detail.html**: fallback 경로로 유지. 삭제하지 않음.
- **mypage.html**: 메타태그 이미 완료 상태 — 할일 목록에서 제거.
- **generate-coin-pages.js 수정 반영 시점**: 스크립트를 고치면 다음 Actions
  실행 시(매일 KST 00:00 또는 수동 Run workflow) 새로 생성된 파일에 반영됨.
  이번 가격 버그 수정은 기존 20개 파일도 직접 일괄 수정해서 즉시 반영함.
- **PAT 권한**: workflow 파일 push는 `Workflows: Read and write`,
  Actions API 트리거는 `Actions: Read and write` 권한 필요.
  Fine-grained PAT는 발급 후 권한 변경 불가 — 필요 시 새로 발급.
- **앞면/뒷면 기준**: 주제 그림(동물·건물 등) = 앞면(`.png`), 인물
  초상·국가 문장 = 뒷면(`-back.png`)
- **스펙 필드 구조**: `year`, `mint`, `country`, `purity`, `weight`,
  `diameter`, `thickness`(선택), `condition`, `verified`
