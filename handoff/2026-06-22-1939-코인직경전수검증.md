# 코인 직경 전수 검증 및 캥거루·버팔로·아웃백 수정 인수인계 문서

작성일: 2026-06-22

## 1. 목표

`coin-descriptions.js`에 등록된 19개 코인 전체의 직경(diameter)·두께(thickness) 값을
조폐국 공식 페이지 및 대형 딜러(APMEX, JM Bullion, Money Metals 등) 기준으로
전수 검증하고, 확인된 오류를 수정한다.

## 2. 현재 상태

**완료 (push 반영 확인)**

- **직경 검증 — 19개 코인 전체 완료** (커밋 `1e9aae2`, `a2daeae`)
  - 검증 결과 표: 일치 16개 / 사용자 확인 후 수정 3개
  - 캥거루: `32.1mm` → **`32.6mm`** 수정 (Perth Mint 공식: 32.60mm)
  - 버팔로: `32.69mm` → **`32.7mm`** 수정 (APMEX/govmint: 32.70mm)
  - 아웃백: `32.1mm` → **`32.6mm`** 수정 (사용자 확인)
  - 그 외 16개(메이플·이글·크루거·판다·브리타니아·필하모닉·성조지·퀸즈라이언·
    라이언이글·네스호·로얄드래곤·브리티시라이언·케이브라이언·말띠·스완·
    체코라이언·레이디저스티스)는 기존 값과 일치 또는 사용자 확인으로 종결

- **레이디저스티스 발행국 재검증** — 지난 세션 수정(니우에) 정확성 재확인 완료
  - Scottsdale Mint Lady Justice 시리즈는 2021~2024년 발행국이 지브롤터(Gibraltar),
    2025년부터 니우에(Niue)로 변경됨을 확인
  - 우리 코드 키워드가 `'2025 레이디저스티스'` 기준이므로 니우에가 맞음 — 지난
    수정이 옳았음을 검증

**미완료 (사용자 액션 필요)**

- **두께(thickness) 값은 이번 세션에서 검증하지 않음** — 직경만 검증 대상으로
  범위를 좁혀 진행했기 때문. 직경 검증 과정에서 다음 사실이 드러났으나 두께는
  아직 코드에 반영하지 않은 상태:
  - 영국 왕립 조폐국(The Royal Mint) "Tudor Beasts" 계열(퀸즈라이언, 로얄드래곤)은
    Money Metals 기준 실제 두께가 **2.7mm**로 확인됨 — 코드 현재값은 2.87mm
    (일반 불리온 규격을 그대로 적용한 것으로 추정, Tudor Beasts는 더 얇음)
  - 성조지·라이언이글·네스호·브리티시라이언은 직경(32.69mm)만 확인됐고
    두께를 알려주는 출처를 못 찾음 — 코드 현재값(2.87mm)이 맞는지 불확실
  - 브리타니아: 공식 두께 2.30mm 확인됨 — 코드 현재값 2.87mm와 다름
  - 캥거루: Perth Mint 공식 두께 2.95mm 확인됨 — 코드 현재값 2.65mm와 다름
    (이번 세션에서 직경만 수정했고 두께는 미수정)
  - 필하모닉: 공식 두께 2.00mm 확인됨 — 코드 현재값 2.37mm와 다름
  - 말띠: 출처별로 2.6mm/2.5mm 등 제각각이라 단일 확정값 불확실
  - 버팔로: 공식 두께 2.95mm 확인됨 — 코드 현재값 2.87mm와 다름
    (이번 세션에서 직경만 수정했고 두께는 미수정)
  - 스완·아웃백·체코라이언·케이브라이언·레이디저스티스는 두께 출처 자체를
    못 찾음

## 5. 참고 사항

- **직경 검증 방법론**: 조폐국 공식 페이지(Royal Mint, Perth Mint, US Mint 등)를
  최우선 출처로 삼고, 공식 페이지에 정보가 없으면 APMEX·JM Bullion·Money Metals·
  govmint 등 다수 대형 딜러에서 일치하는 값을 신뢰. LPM(홍콩)·Truney(대만)는
  봇 차단으로 직접 페이지 접근이 안 되고 검색 스니펫에도 상세 스펙이 거의
  노출되지 않아 이번 검증에서는 출처로 활용하지 못함.
- **같은 1oz인데 직경·두께가 다른 이유**: 금의 밀도(19.32g/cm³)는 고정이라 1oz의
  부피는 항상 같지만, 그 부피를 어떤 직경:두께 비율로 만들지는 조폐국 디자인
  선택의 문제. 따라서 코인마다 비율이 달라도 정상이며, 이 때문에 한 사이트의
  스펙표를 그대로 복붙하면 다른 코인 값이 섞여 들어가는 실수가 발생하기 쉬움
  (레이디저스티스가 실제로 이런 사례였음 — 이전 세션 핸드오프 참고)
- **레이디저스티스 발행국 변천사**: Scottsdale Mint Lady Justice 시리즈는
  2021~2024년 지브롤터(Gibraltar) 발행, 2025년부터 니우에(Niue) 발행으로 변경됨.
  연도가 다른 코인을 검색하면 발행국이 다르게 나오니 검증 시 연도를 반드시
  맞춰서 확인해야 함
- **체코라이언 발행국 주의**: 제조사는 Czech Mint(체코)지만 발행국은
  **니우에**임 — 계속 혼동 주의
- **두께 미확인 항목 표시 패턴**: `coin-descriptions.js`의 `specs` 객체에 `thickness`
  필드를 넣지 않으면 `coin-detail.html`이 자동으로 그 행을 표에서 생략함
  (이전 세션에 적용한 패턴, 체코라이언·케이브라이언에 이미 적용돼 있음).
  두께 재점검 후 값이 확정되면 해당 코인에 `thickness` 필드를 추가하면 자동으로
  표에 다시 노출됨
- **앞면/뒷면 기준**: 주제 그림(동물·건물 등) = 앞면(`.png`), 인물 초상·국가
  문장 = 뒷면(`-back.png`)
- **스펙 필드 구조**: `year`, `mint`, `country`, `purity`, `weight`, `diameter`,
  `thickness`(선택), `condition`, `verified`
- **신규 상품 추가/삭제 시**: `coin-descriptions.js` + `products.js`의
  IMAGE_MAP + 시트 행 3곳을 함께 관리
