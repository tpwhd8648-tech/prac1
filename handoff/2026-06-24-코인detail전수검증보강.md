# 코인 detail 텍스트 전수 검증 및 보강 인수인계 문서

작성일: 2026-06-24

## 1. 목표

`coin-descriptions.js`에 하드코딩된 20개 코인의 `detail` 서술형 설명 텍스트를
공신력 있는 출처(조폐국 공식 사이트, JM Bullion, Money Metals, Atkinsons 등)와
대조하여 사실관계 오류를 수정하고, 부족한 역사·배경 정보를 보강한다.
(specs 필드는 2026-06-22 세션에서 이미 전수 검증 완료 — 이번엔 detail만 대상)

## 5. 참고 사항

- **coin-descriptions.js 수정 시 주의**: `keywords` 배열은 상품명 매칭 식별자이므로 절대 건드리지 않는다.
  Perth Mint 코인(캥거루·말띠·스완·아웃백)이 여러 개이므로 `str_replace` 대신
  Python `content.replace()`를 쓰고 반드시 `keywords` 줄이 포함된 충분히 긴 범위를 지정할 것.
- **Actions 자동 재생성**: coin-descriptions.js 변경 후 push하면
  `generate-coin-pages.yml`이 자동 트리거되어 coin-*.html 20개가 재생성된다.
  `scripts/generate-coin-pages.js`가 동기화되어 있어야 coin 페이지에 반영됨
  (이번 세션 변경 사항은 detail 텍스트 소스가 coin-descriptions.js이므로 generate 스크립트 수정 불필요).
- **"레이디저스티스" detail이 이번에 가장 큰 정보 변화**: 발행국이
  지브롤터(2021~2024) → 니우에(2025~)로 바뀐 중요한 사실이 기존 설명에 없었음.
  현재 사이트에는 2025년판 제품이 등록되어 있으므로 수정 내용이 정확함.
