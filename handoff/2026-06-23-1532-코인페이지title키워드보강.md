# 코인 개별 페이지 title 키워드 보강 인수인계 문서

작성일: 2026-06-23

## 1. 목표

코인 개별 페이지(`coin-*.html`) title에 "투자 금화 구매" 키워드를 추가해
구글 검색 결과에서 거래 의도 검색어("금화 구매" 등)에 더 잘 노출되도록 한다.

## 2. 현재 상태

**완료 (push 반영 및 Actions 트리거)**

- **`scripts/generate-coin-pages.js` title 템플릿 수정** (커밋 `e9b59d9`):
  - 전: `MIDAS BULLION | ${safeName}`
  - 후: `MIDAS BULLION | ${safeName} - 투자 금화 구매`
  - `og:title`도 동일하게 보강

- **GitHub Actions 수동 트리거 완료**:
  - 20개 코인 개별 페이지에 변경 사항 자동 반영 중
  - 예시: `MIDAS BULLION | 2026 메이플 1oz - 투자 금화 구매`

- **SEO 현황 체크 결과** (이번 세션 확인):
  - `contact.html`, `gold-price.html` title/description — 이미 충분한 키워드 포함, 양호
  - 카드 이미지 alt — `products.js`에서 `alt="${name}"` 으로 상품명 자동 삽입, 양호
  - 코인 개별 페이지 title만 보강 필요했고 이번에 완료
