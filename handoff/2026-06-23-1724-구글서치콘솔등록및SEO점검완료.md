## 5. 참고 사항

- **구글 서치 콘솔 소유권 확인 방식**: HTML 태그 방식으로 자동 확인됨.
  `index.html`에 이미 Google 확인 메타태그가 있었기 때문. 태그를 삭제하면
  소유권 확인이 풀리므로 삭제하지 말 것.
- **리치 스니펫 경고 2개 (`shippingDetails`, `hasMerchantReturnPolicy`)**:
  선택사항이며 현재 KakaoTalk 문의 기반 구매 방식에서는 정형화된 정책이
  없어 즉시 추가 불가. 📋 운영/프로세스 항목(구매안내 탭 정책 검토)과
  연동되어 있음 — 정책 확정 후 JSON-LD에 반영하면 됨.
- **`<img>` width/height 불필요 판단 근거**: `style.css` 136번째 줄에
  `.product-img-area { aspect-ratio: 1 }`, 147번째 줄에
  `.product-img-area img { width: 100%; height: 100% }`가 이미 설정되어
  있어 브라우저가 컨테이너 기준으로 공간을 미리 확보함. CLS 개선 효과 없음.
- **네이버 색인 현황**: 현재 메인 페이지 1개만 노출. 코인 상세 페이지들이
  아직 색인 안 됐을 가능성 있음 — 네이버는 JavaScript 렌더링을 잘 못 읽어서
  정적 HTML이 아니면 수집이 늦거나 안 될 수 있음 (이전 세션에서 논의된
  동적 렌더링 문제와 연관). 시간이 지나도 코인 페이지가 안 잡히면 Option A
  (GitHub Actions 정적 생성)가 네이버 대응에도 유효.
- **GitHub PAT**: 이번 세션에서 재발급했다고 확인됨. 만료일은 별도로
  확인되지 않았으므로 다음 push 시 만료 여부 먼저 확인할 것.
