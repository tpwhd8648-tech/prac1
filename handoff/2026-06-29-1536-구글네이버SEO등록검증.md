## 5. 참고 사항

- **구글 사이트맵 "가져올 수 없음" 문제는 GitHub Pages 플랫폼 차원의
  알려진 이슈**임. 향후 같은 증상이 다시 보이거나 다른 사이트맵 관련
  경고가 떠도, 코드/sitemap.xml 파일 자체를 먼저 의심하기보다 실제
  파일이 브라우저에서 정상 열리는지부터 확인할 것 — 파일이 정상이면
  GitHub Pages 쪽 한계로 보고 넘어가도 무방.
- **Search Console 소유자 알림 메일**: 다른 계정으로 같은 사이트를
  인증하면 "새 소유자 추가됨" 알림이 오는 게 정상 동작. 모르는 사람이
  추가된 게 아니라면 무시해도 됨.
- **네이버는 구글과 달리 사이트맵을 정상적으로 인식**하고 있었음 —
  구글 쪽 사이트맵 미인식 문제가 사이트 전체의 검색엔진 친화성 문제는
  아니라는 근거로 참고할 수 있음.
- **"접근 불가한 페이지" 류 진단 항목은 갱신시간을 항상 같이 확인**할
  것. 오래된 갱신시간(예: 6일 이상 전)이면 이미 해소된 과거 이슈일
  가능성이 높으므로, 현재 코드를 grep해서 해당 경로 참조가 실제로
  남아있는지부터 확인 후 대응 여부를 판단할 것.
