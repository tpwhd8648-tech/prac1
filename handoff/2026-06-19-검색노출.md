## 3. 진행 단계

**단계 A. 도메인 구매 (선택, 권장)** — 미진행
- 현재 주소(`github.io/...`)도 기술적으로 검색 등록 가능하지만, 신뢰도와 기억하기 쉬운 정도를 위해 자체 도메인(예: `내사이트.com`) 구매를 권장
- 연 비용 약 1만원~2만원대
- 필수 단계는 아니므로 나중으로 미뤄도 무방

**단계 B. 네이버 서치어드바이저 등록 (필수, 우선 진행)** — 미진행
검색 노출을 위한 핵심 단계. 도메인 여부와 관계없이 지금 주소 그대로 진행 가능.
1. `searchadvisor.naver.com` 접속 후 네이버 아이디로 로그인
2. 웹마스터 도구 → 사이트 등록 → 사이트 주소 입력 (`https://tpwhd8648-tech.github.io/prac1/`)
3. 소유 확인 절차 진행 (네이버가 둘 중 하나의 방식을 제시함)
   - HTML 파일 업로드 방식: 네이버가 주는 인증 파일을 다운받아 레포에 그대로 업로드
   - HTML 태그 방식 (더 간단함): 네이버가 주는 메타태그 한 줄을 `index.html`의 `<head>` 안에 붙여넣기
4. sitemap.xml 제출 (선택, 처음엔 생략 가능, 등록만 해도 크롤링 시작됨)
5. 색인까지 보통 며칠~몇 주 소요

**단계 C. 구글 서치 콘솔 등록 (선택, 추천)** — 미진행
- `search.google.com/search-console` 에서 네이버와 동일한 방식으로 등록 가능
- 네이버 등록 절차에 익숙해진 후 동일하게 진행하면 됨
