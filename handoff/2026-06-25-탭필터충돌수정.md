# 탭 필터 충돌 수정 인수인계 문서

작성일: 2026-06-25

## 1. 목표

이전 세션에서 추가한 당일수령 탭 필터(`applyTabFilter`)가
`loadProducts()` 콜백 내 구형 탭 로직과 충돌하는 문제를 수정한다.

## 2. 현재 상태

### 2-1. 수정 완료

- **문제**: `loadProducts()`(검색 `?q=` 진입 시 호출)가 카드를 새로 그린 후
  구형 로직 `card.dataset.category === filter`로 탭 필터를 적용하고 있었음.
  → "당일수령" 탭이 활성화된 상태에서 검색하면 모든 카드가 다 보이는 버그.

- **수정 내용**: `coins.html` 592번째 줄 부근
  구형 탭 필터 블록(6줄) → `applyTabFilter(activeTab.dataset.filter)` 1줄로 교체.

### 2-2. 전체 충돌/오류 체크 결과 (이상 없음)

| 항목 | 결과 |
|------|------|
| JS 문법 (`node --check`) | 전체 JS 파일 OK |
| `applyTabFilter` 호이스팅 | 동일 `<script>` 블록 내 `function` 선언 → OK |
| `badge-instock` 생성 경로 | SSG(generate 빌드), 동적(`products.js`) 양쪽 동일 클래스명 → 필터 정상 |
| `?instock` + `?q` 동시 진입 | 경로 분리, 충돌 없음 |
| `index.html`, `coin-detail.html` | 탭 로직 없음, 무관 |
| `style.css` | `.ptab`, `.badge-instock` 공용 스타일 자동 적용 |

### 2-3. 변경 파일

- `coins.html` (1줄 수정)
