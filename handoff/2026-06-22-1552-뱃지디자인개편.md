# 뱃지 디자인 개편 인수인계 문서

작성일: 2026-06-22

## 1. 목표

당일수령 뱃지를 IN STOCK / PRE SALE 2종 체계로 개편하고,
색상·위치·크기를 사이트 분위기에 맞게 다듬었다.

## 2. 현재 상태

**완료 (push 후 원격 반영 확인)**

- 뱃지 2종 체계 도입 (커밋 `f118bee`)
  - 기존 `당일수령` 단일 뱃지 → IN STOCK / PRE SALE 2종으로 분리
  - IN STOCK: `available=TRUE` AND `same_day=TRUE`
  - PRE SALE: `available=TRUE` AND `same_day=FALSE`
  - SOLD OUT: `available=FALSE` (기존 오버레이 유지, 뱃지 없음)

- 뱃지 디자인 확정 (커밋 `42889b3`)
  - IN STOCK: 골드 톤 (`#b8860b` / `rgba(184,134,11,0.12)`)
  - PRE SALE: 브론즈 톤 (`#cd7f32` / `rgba(205,127,50,0.12)`)
  - 위치: 이미지 좌상단 (`top:4px; left:4px`)
  - 크기: `font-size:9px`, `padding:2px 7px`
  - 텍스트: 대문자 (`IN STOCK` / `PRE SALE`, 하이픈 없음)
  - 모바일(480px 이하): `font-size:8px`, `padding:1px 4px`, `top/left:3px`

- 적용 파일
  - `style.css`: `.badge-instock` / `.badge-presale` 스타일 정의
  - `products.js`: `createProductCard()` 공용 함수에 조건 분기
  - `coin-detail.html`: `renderDetail()` 함수에 조건 분기

**확인 완료**

- 브라우저에서 뱃지 9개 상품 정상 노출 확인 (사용자 직접 확인)
- 말띠/네스호 order 순서 — 의도된 순서로 종결

## 5. 참고 사항

- **뱃지 CSS 위치**: `style.css`의 `.badge-instock` / `.badge-presale`(기본)와
  `@media (max-width: 480px)` 블록(모바일 축소) 두 곳에 나뉨.
  스타일 수정 시 두 곳 모두 확인할 것.

- **뱃지 노출 조건 요약**:
  - `available=TRUE` + `same_day=TRUE` → IN STOCK (골드)
  - `available=TRUE` + `same_day=FALSE` → PRE SALE (브론즈)
  - `available=FALSE` → SOLD OUT 오버레이 (뱃지 없음)

- **할일 양식**: 색깔 이모지 + 담당자(🤖/👤) 양식은 `docs/PROJECT_INSTRUCTIONS.md`
  섹션 1에 정의되어 있음. 모든 계정에 자동 적용됨.
