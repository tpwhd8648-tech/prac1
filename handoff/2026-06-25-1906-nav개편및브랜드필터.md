# nav 개편 및 브랜드 필터 추가 인수인계 문서

작성일: 2026-06-25

## 1. 목표

coins.html에 브랜드별 필터 드롭다운을 추가하고, nav 항목을 개편하여
사용자가 IN STOCK 상품과 실시간 시세, 브랜드별 상품에 빠르게 접근할 수 있도록 한다.
index.html 브랜드 로고 클릭 시 해당 브랜드 상품으로 바로 이동하는 기능도 추가한다.

## 2. 현재 상태

### 2-1. 완료 — coins.html 브랜드별 필터 드롭다운 (커밋 `0e1ae3d`)

- 기존 탭(금화·당일수령) 옆에 "브랜드별 ▾" 드롭다운 버튼 추가
- 카드에 `data-brand` 속성 추가 (20개 전부)
- 드롭다운 초기화 로직: 카드의 `data-brand`를 스캔해 브랜드 목록 자동 생성
  → 새 코인 추가 시(기존 브랜드라면) 자동 반영됨
- `applyTabFilter()`에 `brand:브랜드명` 케이스 추가
- `scripts/generate-coin-pages.js` 카드 템플릿에 `data-brand` 추가
  → SSG 빌드 시 새 코인도 자동으로 `data-brand` 포함
- `style.css`에 드롭다운 CSS 추가

### 2-2. 완료 — nav 항목 개편 (커밋 `c7fe7ca`)

변경 전: 금화 보기 · 금 시세 · 구매 문의
변경 후: 금화 보기 · IN STOCK · 실시간 시세 · 브랜드별 · 구매 문의

- IN STOCK → `coins.html?instock` (당일수령 필터 자동 적용)
- 실시간 시세 → `gold-price.html` (텍스트만 변경)
- 브랜드별 → `coins.html?brand=1` (드롭다운 자동 열림)
- 데스크탑 nav / 모바일 메뉴 / 헤더 검색 드롭다운 3곳 모두 반영
- `nav.js` 단일 파일 수정

### 2-3. 완료 — 브랜드 파라미터 처리 고도화 (커밋 `d7c7ca7`)

- `?brand=브랜드명` 파라미터 수신 시 해당 브랜드 자동 필터 적용
- `?brand=1` 은 드롭다운만 열기 (nav "브랜드별" 클릭용)
- `urlParams.get('brand')`은 자동 URL 디코딩하므로 `%20` → 공백 변환 처리됨

### 2-4. 완료 — index.html 브랜드 로고 링크 연결 (커밋 `d7c7ca7`, `1cfbfba`)

- 브랜드 로고 11개 전부 클릭 가능하도록 `<a>` 태그로 감쌈
- 상품 있는 8개: `coins.html?brand=브랜드명` → 해당 브랜드 자동 필터
- 상품 없는 3개(PAMP·Heraeus·New Zealand Mint): `coins.html` → 전체 상품 보기

### 2-5. 자동/수동 적용 범위

- 기존 브랜드로 새 코인 추가 → SSG 빌드 시 자동 반영 (data-brand 자동 포함, 드롭다운 자동 등장)
- 새 브랜드 추가 → index.html 브랜드 로고 수동 추가 필요

## 5. 참고 사항

- **브랜드 필터 URL 규칙**: `coins.html?brand=Royal%20Mint` 형태. `urlParams.get('brand')`이
  자동 디코딩하므로 코드에서 별도 처리 불필요. `data-brand` 값과 정확히 일치해야 함.
- **드롭다운 브랜드 목록은 카드 스캔으로 자동 생성**: `initBrandDropdown()` 함수가
  DOM의 `data-brand`를 읽어 목록을 만드므로, SSG 빌드 후 새 브랜드가 카드에 있으면
  드롭다운에 자동 등장함. 별도 목록 관리 불필요.
- **헤더/히어로/배너 디자인 시안**: 이번 세션에서 A안(화이트)/B안(다크)/C안(풀블랙)
  3가지를 mockup으로 제작해 비교했으나 방향 미확정. 다음 세션에서 계속.
